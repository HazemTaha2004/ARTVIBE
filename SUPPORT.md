# Support Guide

Thank you for using ArtVibe! If you run into problems or have questions, here are the best channels to get help:

---

## 🙋‍♂️ Question or Discussions
For general questions, implementation queries, or design discussions, please open a thread in the **GitHub Discussions** tab *(or open a GitHub Issue)*.

## 🐛 Bug Reports & Feature Requests
If you've identified a bug or want to suggest a new feature, please open a GitHub Issue:
1. Ensure the issue hasn't been logged previously by checking the active Issues search.
2. Provide a clear description, reproduction steps, browser and OS details, and expected outcome.
