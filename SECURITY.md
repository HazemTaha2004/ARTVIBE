# Security Policy

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |

## Reporting a Vulnerability

We take the security of this project seriously. If you find a security vulnerability or concern, please do not open a public issue. Instead, report it privately using the steps below:

1. **Email us** at [security email placeholder] with details of the vulnerability.
2. Provide a description of the vulnerability, instructions on how to reproduce it, and the potential impact.
3. We will acknowledge your email within 48 hours and work on a fix as quickly as possible.
4. We will coordinate a release and announce the vulnerability publicly once the fix is applied.
